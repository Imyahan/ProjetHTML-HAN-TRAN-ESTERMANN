Projet

Ce projet a été réalisé dans le cadre du module TI504N – Programmation Web. Il avait pour objectif principal la conception d’un site vitrine du département d’informatique de l’EFREI, permettant de présenter de manière claire et structurée les informations essentielles du département.

Le site a été développé exclusivement en HTML5, CSS3 et JavaScript, sans utilisation de frameworks ni de bibliothèques externes, conformément aux consignes. Une attention particulière a été portée à la qualité du code, à la structure des pages ainsi qu’au design responsive, afin d’assurer une navigation fluide sur tous les supports.

Ce projet a également permis de développer le travail collaboratif au sein du groupe, notamment grâce à l’utilisation de Git et GitHub pour la gestion des versions, le partage du code et le suivi des contributions de chacun.
